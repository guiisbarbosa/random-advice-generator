<h1>Gerador de conselhos aleatórios</h1>
<p>O Gerador de conselhos aleatórios é um projeto retirado do site FrontEnd Mentor, desenvolvido como exercício para o curso de Desenvolvimento Web FrontEnd em que estou estudando chamado, DevQuest.</p>

<p>Parte do projeto foi feita algum tempo atrás como exercício dos módulos de HTML e CSS, em que existia apenas uma página estática. Agora como parte de aprendizagem dos módulo de JS Avançado, foi pedido para gerar os conselhos de forma aleatória através de um API (https://api.adviceslip.com).</p>
